# Reducer

Map MathJs parse tree to LISP and run LISP
All syntatic sugar can be mapped to LIST. Therefore MathJs becomes a complete functional programming language

## Installation

```sh
npm install
```

## Build

- Build: `npm run build`
- Clean: `npm run clean`
- Build & watch: `npm run start`

## Run

```sh
node src/Demo.bs.js
```
